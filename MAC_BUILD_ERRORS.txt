Sprout Mac Build Errors
=======================

Use this file to hand Mac/Xcode build problems back into the repo.

How to use it on the Mac:
1. Build in Xcode.
2. Open the Issue Navigator with Command + 5.
3. Paste the full error text here.
4. If the list is very long, include:
   - the first 5 full errors
   - one example of each different error type
5. Commit and push this file to `main`.

Suggested format:

Date:
Xcode version:
Device or simulator:

Build errors:
- Paste errors here

Notes:
- Add anything useful about what you clicked or changed before the build failed.
