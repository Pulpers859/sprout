# Docs

Support material is grouped here so the repo root stays focused on the live app surfaces.

- `project-handoff.md`
  Repo-specific source-of-truth, branch, and workflow notes.
- `ai-project-handoff-template.md`
  Reusable handoff template for similar local app repos.
- `mac-build-errors.md`
  Paste-in template for Mac/Xcode build failures that need to come back into the repo.
